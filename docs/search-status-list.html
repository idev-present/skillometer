Search Status List

Метод: GET /dict/search_status

Описание:

Этот метод предназначен для получения списка статусов резюме соискателей. Возвращает массив объектов, каждый из которых представляет собой статус поиска работы.

/dict/search_status

Responses

[ { "id": "active", "name": "В активном поиске", "habr_id": null }, { "id": "not_active", "name": "Не ищу работу", "habr_id": null }, { "id": "passive", "name": "Рассмотрю предложения", "habr_id": null } ]
